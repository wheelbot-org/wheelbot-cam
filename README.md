# WheelBot.org Cam

Camera module for Wheelbot.org project. Stable FPS - 26 with VGA (640x480) frames.

## Comments

This project was tested on the ESP32-D0WDQ6 chip version. Other versions may require some adaptation. To check your chip version, run:

```bash
~/.platformio/packages/tool-esptoolpy/esptool.py --chip esp32 --port /dev/ttyUSB0 flash_id
```
